## Tính năng chính

- Đăng ký/đăng nhập bằng **số điện thoại**

- Chuyển tiền theo **số tài khoản** + UI tự lookup **tên người nhận**
- UI được chỉnh lại đẹp/clear hơn (React + Tailwind)
